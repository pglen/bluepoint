BluePoint Encryption is a simple, secure, state of the art block encryption algorithm. It requires minimal processing power and has a low memory footprint. The algorithm produces an even bit distribution, even when its fed with small data or homogenus content. This makes it highly resilient against crypt analysis, and an ideal candidate where safe, secure and customized encryption is needed. The BluePoint Encryption is available in source code format in the following languages: 'C', PHP, Perl, JavaScript, ... All the source code ports are cypher text compatible, content encoded with one port, can be decoded in other ports.

Bluepoint Encryption Study

   In testing Bluepoint, one of the exceptional properties we found was that in the clear text change, the change propagated to every byte of the cyphertext, with no predictable bit distribution. We fed the Bluepoint algorithm various clear text samples and examined the cypher text. Below, is what we found.

Original Context, this is what we used as a starting point:

org=aaaaa  pas=1234
ENCRYPTED:
-9c-7b-db-24-52
END ENCRYPTED

Changed (added) one byte clear text to the Original Context, cypher text has changed completely:

org=aaaaaa  pas=1234
ENCRYPTED:
-d6-ff-a9-04-c4-bc
END ENCRYPTED

Changed (added) one byte to the password of Previous  Context, cypher text has changed completely:

org=aaaaaa  pas=12345
ENCRYPTED:
-d8-b3-af-d4-ae-70
END ENCRYPTED

Changed one bit of the password of Previous Context, cypher text has changed completely:

org='aaaaaa'  pas='12345'
ENCRYPTED:
-f4-97-93-02-2a-0c
END ENCRYPTED

Changed one bit of the clear text of Previous Context, cypher text has changed completely:  (note a=0x41 b=0x42)

org='baaaaa'  pas='12345'
ENCRYPTED:
-94-8b-12-f1-08-c7
END ENCRYPTED

Moved the one bit change of the clear text of Previous Context, cypher text has changed completely:

org='aaaaab'  pas='12345'
ENCRYPTED:
-d4-ed-39-b5-60-35
END ENCRYPTED

Changed one letter casing (see: Jumped) of the clear text, the cypher text has changed completely:

org='The quick brown Squirrel Jumped over the Fat Cat'  pas='1234'
ENCRYPTED:
-d6-03-a2-43-c0-26-33-ec-de-18-f6-7c-3f-73-fa-4f-bd-1a-34-8b-a0-2e-f3-e0-8c-64-2e-f9-5a-98-7e-c3-af-d7-9a-ad-21 -4f-ba-da-d8-ab-60-6b-93-dd-8f-7c
END ENCRYPTED

org='The quick brown Squirrel jumped over the Fat Cat'  pas='1234'
ENCRYPTED:
-ca-95-f8-02-ac-b7-59-a5-e9-7d-87-92-f5-2e-56-ce-91-98-f7-e7-20-22-66-53-ee-b4-38-3b-82-9d-1f-f7-36-a8-b4-f0-89 -5c-5c-0f-7f-80-1b-c2-7e-3b-5b-f5

Conclusion:

   When we inputted monolithic text into the BluePoint Encryption Algorithm, the cypher text varied greatly when we changed ONE byte in the clear text, and varied just as greatly  when we changed ONE bit. This makes the BluePoint algorythm resistive to crypographic analisies.
